# 🗺️ 智能AI旅行管家 (Smart AI Travel Agent)

基于 **LangChain + Pydantic 强约束架构** 与 **Streamlit** 前端构建的智能旅行规划专家。系统深度集成了本地 RAG（检索增强生成）私有小红书避坑知识库，并联动了实时天气、预算风控网关以及强大的大模型输出自动修复机制。

---

## ✨ 核心亮点

* **🧭 状态驱动的长对话记忆核心** 利用 Streamlit 的 `session_state` 配合定制的 `extractor_chain`，能够自动从多轮碎片的口语对话中捕获目的地、游玩天数、预算上限及偏好标签。在关键约束（如目的地、天数）补全前自动拦截，避免无效规划。
* **📚 RAG 私有干货与避坑指南动态注入** 对接 **Chroma 向量数据库** 与智谱 `embedding-3` 模型。在规划行程时，系统会自动将用户的“目的地 + 需求”进行关键词净化，智能匹配并提取本地小红书级别的私有攻略、避坑干货（如“别坐西湖 150/小时的私家摇船”），作为硬性限制喂给大模型。
* **🚨 实时双重风控网关（天气 + 价格）** 深度绑定城市天气 API 与预算 Baseline 估算模型。当测算出有降雨预期或行程开销可能**超出**用户预期预算时，系统会自动触发看板的高亮风控警告（红标拦截），提供真正的智能化风控提示。
* **🛠️ 智能格式对齐与自动修复网关** 由于前端依赖 Pydantic 进行严格的规整卡片看板渲染，系统内置了格式容错机制。当大模型首次输出的 JSON 格式因特殊字符或幻觉损坏时，系统会**自动捕获异常并启动二次修复链**，严厉校准格式，确保系统 100% 稳定不卡死。
